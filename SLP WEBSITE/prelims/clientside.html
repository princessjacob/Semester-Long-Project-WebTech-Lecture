DEFINITION

  • Allows program (i.e. scripts) to be downloaded from a web server and executed in the client environment (e.g. browser)

JAVASCRIPT

It was invented in 1995 by Brendan Eich and in 1997 it became an ECMA standard. It is designed for creating network-centric applications.

ECMASCRIPT

-European Computer Manufacturer's Association. -It is a subset of JavaScript. -It is a Standard for a scripting languages

JSCRIPT

Microsoft dialect of the ECMAScript scripting language specification. The modern scripting language with a wide variety of applications is the JScript 10.0.

JAVA APPLETS

It is a small application which is written in Java or another programming language.

ACTIVEX CONTROLS

It is a component program object that can be re-used by many application programs within a computer or among computers in a network.

FLASH ANIMATIONS

It is also called as Flash cartoon. It is an animated film that is created with the Adobe Flash.

MICROSOFT SILVERLIGHT

Integrates multimedia,graphics,animations, and interactivity into a single run-time environment.

ADOBE INTEGRATED RUNTIME(ADOBE AIR)

It is a cross-operating system runtime.It lets the developers combine HTML,JavaScript,Adobe Flash and Flex technologies, and ActionScript.

DYNAMIC (X)HTML

Page embelishments, visual "effects", content generation and manipulation, user interaction, document and page navigation

CLIENT-SIDE FORM DATA VALIDATION

Checking a web form and making sure all the data are complete and correct before sending off to the server.

BROWSER SUPPORT

No scripting support, Scripting disabled, Plugin availability

VERSION INCOMPATIBILITIES, NON-STANDARD IMPLEMENTATIONS

CAPABILITIES RESTRICTIONS

SECURITY RISKS

Browser implementation defects (e.g. buffer overflows).Example: Javascript, cross-site scripting (XSS) or cross-site request forgery

(XSRF) ISSUES

Malicious ActiveX controls
