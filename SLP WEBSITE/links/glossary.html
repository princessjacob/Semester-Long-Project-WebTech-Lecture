ARCHITECTURE

A network characteristic that can be broadly classified as peer-to-peer or client/server architecture.

BLOCK DEVICE DRIVERS

Device drivers that supports a file system which is called as the block devices.

CHARACTER DEVICE DRIVERS

Device Drivers that performs the Input/output in the form of byte stream.

DEVICE DRIVERS

Device Driver is a program which controls a specific type of device and is attached to the computer. It is a bit of software which tells the (OS) Operating System in what way to communicate with the hardware.

DIGITAL CERTIFICATES

Digital certificates are used to verify that a user sending a message is who he or she claims to be, and to provide the receiver with the means to encode a reply.

GOPHER

A protocol that has an ability to view and extract documents from the Web that are stored on remote web servers. It is designed to access a Web server or database with the use of internet. It was discussed in RFC 1436.

HTTP TRAFFIC

The amount of data sent and received by visitors to a website.

INTERCONNECTIONS FROM COUNTRY TO COUNTRY

The satellite link internet connection to an ISP is now also available in rural areas where internet services are sometimes isn’t available or limited. The uplink connection from the LAN to the ISP is usually by a telephone line/modem connection and isn‘t as fast as the satellite downlink.

INTERNET ASSIGNED NUMBERS AUTHORITY (IANA)

IANA is a department of ICANN(Internet Corporation for Assigned Names and Numbers) and is tasked to control numbers for protocols, the Country Code Top Level Domains and maintains the IP Address allotments.

INTERNET OF THINGS

The Internet of Things or sometimes known as Internet of Everything (IoE) talks about how our devices connect with each other, this is called as machine-to-machine and also how our devices connects to the internet. IoT also uses cloud-based applications without it, it will not work.

INTERNET SERVICE PROVIDER

ISP or Internet Service Provider is used by smaller companies and individuals that provide connectivity to the internet.

MEMORY MAPPED DEVICE DRIVERS

Device drivers which has direct access to the device memory.

NETWORK

Computer network or network is well-defined as a group or collection of computer systems that are linked together. It is also called the data network. It is the series of nodes, physical device, that is interconnected by communication paths in order to transmit, receive and exchange data voice and video traffic.

ORIGIN SERVER

A Web server that contains the original Web page.

PACKET SNIFFER

A computer program or hardware that can intercept and log traffic that passes over a digital network or part of a network. This is also called a packet analyzer.

PROTOCOLS

Protocols are communication language between network devices and it is sometimes indicates to as an access method.

PROXY SERVER

A server that acts as an intermediary for requests from clients seeking resources from other servers.

SOCKET | NETWORK SOCKET

Socket is a one endpoint of a two-way communication link between two programs running on the network.

TOPOLOGY

A network characteristic that is the geometric sequence of the computer system.

TRANSMISSION CONTROL PROTOCOL/INTERNET PROTOCOL (TCP/IP)

A protocol used over the internet in order for the computers to communicate to each other. It is also used in a private network as a communication protocol. It is also used in a private network as a communication protocol. TCP/IP is a consists of two-layer program.

UNIFORM RESOURCE ALLOCATOR (URL)

URL or Uniform Resource Locator is a type of URI or Uniform Resource Identifier. URL is also termed as “web address”. URL is used to locate a resource on the web.

USENET

A collection of newsgroups where the users can post messages. The following messages are then stored in different servers. These messages are only available for a certain amount of time.

WEB CRAWLERS | WEB SPIDERS

Web crawlers, sometimes called a spider, is an Internet bot which systematically browses the World Wide Web. Web spiders are typically for the purpose of Web indexing (web spidering). This is often used in an automated manner.

WIDE AREA INFORMATION SERVICES

WAIS which is pronounced as “ways” or Wide Area Information Servers. It uses indexing or collecting information from different information servers and searching document indexes. It has it’s own Internet protocol. WAIS also has four parts, the client, server, database and the protocol. It is based as search model of information rather than browsing.

WIRED INTERCONNECTIONS TECHNOLOGY

It is a way to transmit data which uses wires to communicate between computers. It is said to be the most stable in all the types of communication services.

Wireless Interconnection Technology

A way to transfer information within devices/ points that are not connected by any conductor. Information is transferred trough radio waves.
