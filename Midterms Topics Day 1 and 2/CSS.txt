CSS
CSS or Cascading Style Sheets is used to control the design of a web page. It is used in formatting the style or design of the elements/ attributes that has been implemented in the html. In other words, CSS instructs the content of a webpage on how should it look. It is the presentation aspects of structural makup documents like HTML. It was developed by Håkon Wium Lie and Bert Bos.
Example of CSS below:
body {
font-family: Times New Roman;
color: red;
font-size: 16px;
}
