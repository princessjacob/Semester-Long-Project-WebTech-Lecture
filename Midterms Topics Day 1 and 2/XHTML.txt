XHTML - Extensible Hypertext Markup Language
      - Reformulation of XHTML in XML 

Versions
	- XHTML 1.0 
		-In December 1998, the working draft was entitled Reformulating 
HTML in XML. And it was changed into XHTML 1.0: The Extensible Hypertext Markup
Language in February 1999.

		*XHTML 1.0 Strict - includes elements and attributes that have
not been marked deprecated in the HTML 4.01 specification.
		
		*XHTML 1.0 Transitional - includes the presentational elements

		*XHTML 1.0 Frameset -  allows for the definition of frameset 
documents. It is a common web feature in 1990's

	- XHTML 1.1
		-In 1999, the W3C released a first draft.
		-One of the two first implementations of modular XHTML

	- XHTML 1.2
		-An extension for XHTML 1.1 that uses RDFa, Access, and Role

	- XHTML 2.0
		-HTML frames was replaced by XFrames
		-A general-purpose semantic markup language designed with extension in mind.

	- XHTML 5.0
		-It adds many new elements
		-Such as section and aside tags
