DOM Level 1

It was introduced in 1998 by W3C.
Provided a complete model for an entire HTML or XML document, it
includes to change any portion of the document.
It defines <html> tag as a special element node, called as the document 
element. The most common object types of DOM Level 1 are document, element, attr
and text.
The DOMImplementation is an object class that provides access to Document creation 
methods.
