NAVIGATOR Object
Navigator is the object that holds or contains browser's information.

NAVIGATOR OBJECT PROPERTIES
appCodeName 
	The appCodeName property pertains to a browser's code name, this property is 	considered  as deprecated because it was removed from the standards of Web, but 		other browsers maybe still support this property since it is being processed to be 	dropped.
		Syntax:  x = window.navigator.appCodeName
appName
	This property returns the name of the browser, it is also considered as  deprecated. 
		Syntax:  x = window.navigator.appName
appVersion
	This property returns a string value of the information of the version of the browser. 	This property is also considered as a deprecated.
		Syntax:  window.navigator
appMinorVersion
	This property returns a string value of the browser's minor version.
		Syntax: navigator.appMinorVersion
cpuClass
	It returns the user's operating system's CPU class.
		Syntax: navigator.cpuClass
mimeTypes
	It returns a list of the browser's supported MIME types.
		Syntax: navigator.mimeTypes
opsProfile
	This property is only visible when the object navigator is placed in an enumerating  for loop.
		Syntax: navigator.opsProfile
systemLanguage
	
cookieEnabled
	It determines if the cookies in the browser are enabled or not.
		Syntax: navigator.cookieEnabled
geolocation
	Returns an object of geolocation that can be used to know the position of the user.
language
	Returns the browser's language.
		Syntax:.navigator.language
onLine
	It determines if the browser is online or not.
		Syntax: navigator.onLine
platform
	Returns the platform on which the browser is compiled.
		Syntax: navigator.platform
product
	Returns the browser's engine name.
		Syntax: navigator.product
userAgent
	Returns the header of the user-agent sent to the server by the browser. It is also considered as deprecated.
		Syntax: navigator.userAgent
NAVIGATOR OBJECT METHODS
javaEnabled()
	It specifies if the browser has a Java that is enabled or not.
		Syntax: navigator.javaEnabled()
taintEnabled()
	It specifies if the browser has a data tainting that is enabled or not.
		Syntax: navigator.taintEnabled()
