DEVICE DRIVERS

Device Driver is a program which controls a specific type of device and is attached to the computer. It is a bit of software which tells the (OS) Operating System in what way to communicate with the hardware. It is said to be a small program which allows a computer and a device to connect with one another. Its purpose is to hold the requests which is made by the kernel with regard to a specific type of devices. It resides within the Digital UNIX kernel. Block Device Drivers - supports a file system which is called as the block devices. Character Device Drivers - it performs the Input/output in the form of byte stream. Memory Mapped Devices - it is the device driver which has direct access to the device memory.
