Element-Specific Attributes

data-* attributes

Attributes in data-* shouldn’t contain uppercase letters. It is used to make the data of a web page or web application be private. Example syntax of the attribute is <element data-* = ”value”>. data-* attributes or custom data- attributes simplifies the storage of the application data in web page or web application.

Role, ARIA-* Attributes

Accessible Rich Internet Applications or ARIA is a way to make the content of a web page or web applications be used or understand by the people with disabilities. 
Roles that are included to make it clearly part of WAI-ARIA Role taxonomy.
1. application
2. banner
3. complementary
4. contentinfo
5. form
6. main
7. navigation
8. search
