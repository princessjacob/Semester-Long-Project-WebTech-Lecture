JavaBean - is a specially constructed Java class written in the Java and coded according to the JavaBeans API specifications.

	1. It should have a default, no-argument constructor.
	2. It should be Serializable.
	3. It should provide methods to set and get the values of the properties, known as getter and setter methods.
