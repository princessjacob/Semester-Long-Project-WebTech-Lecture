SCREEN
The object screen holds or contains the visitor's screen information.

SCREEN OBJECT PROPERTIES
width
- this property returns the screen's total width
	Syntax: 
		 width   =  window.screen.width;
pixelDepth
- return a bits per pixel of the screen's color resolution
	Syntax:
		 pixel   =  window.screen.pixelDepth ;
height
- return the screen's total height.
	Syntax:
		 height   =  window.screen.height ;
colorDepth
- it return the color palette for displaying images' bit depth
	Syntax: 
		 color  =  window.screen.colorDepth ;
availWidth
-it returns the the screen's width except the taskbar of the window
	Syntax: 
		 availwidth   =  window.screen.availWidth ;
availHeight
- it returns the screen's height except the taskbar of the window
	Syntax:
		 availheight   =  window.screen.availHeight ;
