CSS FRAMEWORKS
CSS Frameworks are used to make work easier betweek cross-device and cross-browser compatibility. It makes sure that our website will look exactly the same on our devices when we try to access it. Examples of CSS Frameworks are Pure, Bootstrap, Base, Skeleton and many more frameworks. 

These are some common CSS Frameworks: 
a. Pure - is a frameworks that only include stylesheet files. Pure is a smaller framework that the all set of modules weighing in at only 4.3KB for the one file when gzipped or minified. The modules of Pure framework are Grids, Buttons, Base, Menus,Forms, and Tables.
b. Bootstrap - this framework is the most popular and most used CSS framework among all the other CSS frameworks. "Sleek" is what Bootstrap call itself.  It was created by Mark Otto and Jacob Thornton.
c. Base - this framework also a responsive CSS framework, not only in CSS but also in HTML. It has a feature it which it ensures that the website will work with all kinds of devices whether it is big or small.
d. Skeleton - this framework provides minimal design for websites. Its modules are Buttons, Forms, Layouts and Typographies
