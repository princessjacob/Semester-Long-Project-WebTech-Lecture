TCP/IP: TCP/IP or the TRANSMISSION CONTROL PROTOCOL/INTERNET PROTOCOL, is the protocol used over the internet in order for the computers to communicate to each other. It is also used in a private network as a communication protocol. TCP/IP is a consists of two-layer program. First, transmission control protocol which manages and assembles a file into packets in which it is transmitted on the internet and it is received by the TCP layer and reassembles it on the original form of the message. Second, is the internet protocol which holds the address part of every packet in order for the packet to get in the correct destination.
