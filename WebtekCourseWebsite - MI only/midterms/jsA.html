Definition

It is a global object. The window object is the primary object of a browser and represent the browser's window.

Properties

  • window.alert()

    • It is a method that displays an alert box with a message in it. The use of this is to make sure that the information comes through the user.

  • window.open()

    • It is a method that opens a new browser. And the .close() is a method to close the window.

  • window.setTimeout()

    • It is a method that evaluates an expression after a specified number of milliseconds. And the clearTimeout() is a method to stop it from running.

  • window.setInterval()

    • t is a method that evaluates an expression after at specified intervals in milliseconds. It also continues calling the function until clearInterval() is called.

  • window.encodeURI()

    • It is a function that is used to encode a URI. To decode an encoded URI, use the decodeURI().

  • window.innerWidth

    • Get the computed inner width for the first element (include the padding but not the border). It is a method that returns the width of the element.

  • window.localStorage

    • It allows you to access a local Storage Object. And it has no expiration time.

  • window.history

    • It is an object that can be written without the window prefix. To load the previous URL in the history list, use history.back(). To load the next URL in the history list, use history.forward().

  • window.location

    • It is an object that can be written without the window prefix. Here are some examples:

    a. .href()

    returns the URL of the current page

    b. .hostname()

    returns the name of the internet host of the current page

    c. .pathname()

    returns the pathname of the current page

    d. .protocol()

    returns the web protocol of the page (http: or https:)

    e. .asign()

    it is a method that loads a new document
